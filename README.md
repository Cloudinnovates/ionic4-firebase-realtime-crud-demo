# ionic4-firebase-realtime-crud-demo
Ionic 4 Firebase Realtime CRUD demo with email authentication login and social login

## Get your google firebase credentials, from firebase console (web setup)
```
const config = {
    apiKey: "",
    authDomain: "",
    databaseURL: "",
    projectId: "",
    storageBucket: ""
};
```
